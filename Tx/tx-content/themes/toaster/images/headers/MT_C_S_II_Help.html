Read First

The Basic_MT_C_S_II.zip file contains the following files:

  1. Compile_C_S.bat - a batch file that when double clicked will compile the MultiThreaded Client and Server programs.
  2. Helloworld.html - a HTML file that acts as a client and makes a request of the running server.
  3. HTTPGetNames.html - a HTML file that acts as a client and makes a request of the running server.
  4. HTTPGetServlet.html - a HTML file that acts as a client and makes a request of the running server.
  5. MT_C_S_II_Help.html - this file provides help on how to get started.
  6. tcpClient.java - the Java source file for the tcpClient.java program.
  7. tcpServer.java - the Java source file for the tcpServer.java program.
  8. MT_C_S_II_Tutorial.html - a file that describes how to build the MultiThreaded Client Server II program.
  9. run_Javadoc.bat - the batch file that executes the Javadoc file for the tcpClient.java and tcpServer.java files.
  10. Start_Server.bat - when double-clicked will start the execution of the tcpServer.java program.
    • When running this application, this program should always be started first.
  11. Start_Client.bat - when double-clicked will start the execution of the tcpClient.java program.

How to compile and run

There are two ways to run the programs found in this .zip file.

  1. Multithreaded client Server program:
    1. You can double-click on the Compile_C_S.bat file that is supplied with the Basic_MT_C_S_II.zip file.
    2. You can load the tcpServer.java and tcpClient.java files into an IDE and run the programs from that environment.

If you do not have the Java documentation

Like all catalysts, enzymes increase the rate of a reaction by lowering its activation energy (Ea‡). As a result, products are formed faster and reactions reach their equilibrium state more rapidly. Most enzyme reaction rates are millions of times faster than those of comparable un-catalyzed reactions and some are so fast that they are diffusion limited. As with all catalysts, enzymes are not consumed by the reactions they catalyze, nor do they alter the equilibrium of these reactions. However, enzymes do differ from most other catalysts in that they are highly specific for their substrates. Many drugs and poisons are enzyme inhibitors. Activity is also affected by temperature, pressure, chemical environment (e.g., pH), and the concentration of substrate. Some enzymes are used commercially, for example, in the synthesis of antibiotics. In addition, some household products use enzymes to speed up biochemical reactions (e.g., enzymes in biological washing powders break down protein or fat stains on clothes; enzymes in meat tenderizers break down proteins into smaller molecules, making the meat easier to chew).
